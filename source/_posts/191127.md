#### hexo disqus 적용
_config.yml 파일수정
* 주의 : _config.yml 파일은 2개가 존재한다 
- hexoblog>_config.yml
- hexoblog>themes>[테마]>_config.yml
둘다 수정을 해줘야 잘 작동함

```markdown
comment:
    # Name of the comment plugin
    enable: true
    type: disqus
    shortname: kindfamily-1     # (required)

```
